# Mercado Libre | Coupons

Save time and money on your purchases! **Mercado Libre | Coupons** is a Google Chrome extension that allows you to automatically apply all available coupons on Mercado Libre Argentina with a single click.

## üöÄ Features

* **Automatic Application:** Automatically scans coupon pages and activates them for you.
* **Easy to Use:** Simple and intuitive interface. Just press a button and let the extension do the work.
* **Smart Navigation:** Automatically detects pagination and moves to the next page of coupons until the process is complete.

## üõ†Ô∏è Installation

Since this is an extension in development (or for personal use), you must install it manually in Chrome using developer mode:

1. **Download or clone** this repository to your computer.
2. Open Google Chrome and go to `chrome://extensions/`.
3. Enable **"Developer mode"** in the top right corner.
4. Click the **"Load unpacked"** button.
5. Select the folder where you saved/cloned this project.
6. **Done!** The extension should now appear in your extensions bar.

## üìñ Usage

1. Click the extension icon in the Chrome toolbar.
2. Press the **"Apply all coupons"** button.
3. A new tab will open, redirecting you to the Mercado Libre coupons section.
4. The extension will begin claiming coupons automatically and navigating through pages on its own.
5. Once finished, an alert will appear indicating that the process is complete.
